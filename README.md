# ContractGuard 🛡️

AI that flags dangerous clauses in freelance contracts.

## What it does

Upload any freelance contract (PDF or DOCX). Claude AI analyzes it clause by clause and flags HIGH/MEDIUM/LOW risk items with rewrite suggestions.

- ⚡ Results in 3 minutes
- 🔒 Your contract is never stored
- 💰 ~$0.05 per analysis (your own API key)
- 📄 PDF report for $2.99
